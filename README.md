## 📋 What is MARs?

MARs stands for Multi-Agent Research System. It is an AI tool designed to help improve how we check and manage scientific knowledge. MARs questions ideas more deeply, spots errors better, and keeps track of valid research more clearly. This system supports researchers who want more reliable results.

You do not need to know anything about programming to use MARs. This guide shows you how to download, install, and run MARs on your Windows computer.
